De evaluatie

Een kritische beschouwing van mijn functioneren

Deze stage is me over het algemeen goed bevallen en heb ik als zeer leerzaam ervaren. Nu ga ik met veel meer kennis en een verbeterde instelling terug naar school.
Wat ik als zeer positief heb ervaren aan deze stage is dat het zo gevarieerd was. Ik heb meegewerkt aan meerdere projecten, in verschillende programmertalen en heb ik vaardigheden opgedaan voor het duidelijk en overtuigend overbrengen van ideeën. Daarbij ben ik handiger en sneller geworden met verschillende programma’s, vooral in de linux omgeving.
Ik denk vanuit mijn eigen ervaring dat deze stage wel echt het uiterste van je vraagt en niet iedereen bij dit kantoor stage zou kunnen lopen. Toch was ik zelf erg tevreden met de geboden uitdaging en het elke keer gepusht worden voor een stapje beter.
Wat ik ook bijzonder vond aan deze stage is het mijn stage erg afwisselend geweest en kreeg ik veel verantwoordelijkheid, wat ik als zeer positief heb ervaren. Ik heb ook geleerd om sneller en efficiënter te werken.

Bereikte leerdoelen

Mijn belangrijkste leerdoel was zoveel mogelijk kennis te maken en te werken met verschillende programma's en ik denk dat ik mijn doel heb bereikt.

Kennis en vaardigheden uit mijn studie die ik hebt kunnen benutten

Tijdens mijn opleiding op school heb ik met de intellij IDEA en git gewerkt en ik heb ook css, html en java geleerd. Ik was in staat om die vaardigheden te gebruiken tijdens mijn stage.

Nieuwe kennis en vaardigheden die ik heb opgedaan

Tijdens deze stage leerde ik de volgende dingen:
Scrum/Jira, Principles of Scrum, Maven commands, Pom's, Maven repositories, Maven Projectstructuur, Visual Studio Code, Typescript, Angular CLI, Angular componenten, JSON, HTTP Client, kafka simple, kafka stream, ktable en Ksql, confluent cli. Ik heb ook geleerd hoe kan je in een Fedora linux-omgeving gaan werken, en hoe je je PATH kunt permanent instellen.
